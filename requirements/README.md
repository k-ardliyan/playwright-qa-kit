# Requirements

Folder ini untuk **file fitur** yang Planner baca. Template & meta file pakai prefix `_`.

1. Salin [`_TEMPLATE.md`](_TEMPLATE.md) → `nama-fitur.md`
2. **Contoh latihan pipeline:** [`example-login-extension.md`](example-login-extension.md) (sudah valid, 2 skenario otomatis + 1 @manual)
3. Panduan: [docs/README.md](../docs/README.md) · [docs/GUIDE.md](../docs/GUIDE.md) · Format: [docs/writing-requirements.md](../docs/writing-requirements.md)
4. Validasi: `npm run validate:requirement -- requirements/nama-fitur.md`
