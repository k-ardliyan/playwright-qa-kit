# Requirements

Folder ini untuk **file fitur** yang Planner baca. Template & meta file pakai prefix `_`.

1. Salin [`_TEMPLATE.md`](_TEMPLATE.md) → `nama-fitur.md` (folder ini, sejajar template)
2. Contoh: [`example-login-extension.md`](example-login-extension.md)
3. Panduan: [docs/GUIDE.md](../docs/GUIDE.md) · Format: [docs/writing-requirements.md](../docs/writing-requirements.md)
4. Validasi: `npm run validate:requirement -- requirements/nama-fitur.md`
